# Gerador de Nota Fiscal DANFE - Universal Parser

Este é um interpretador universal de documentos fiscais XML em Python.
Ele converte arquivos XML (NF-e, NFC-e, Eventos e Inutilização) em comprovantes e espelhos em formato PDF, de forma automática e resiliente contra problemas de encoding e falhas de estrutura XML.

## Tecnologias
* Python
* CustomTkinter
* FPDF
* brazilfiscalreport
* ElementTree XML API
